# 2FA
## Chrome 和 Edge
* 2025-01 记录。
* **2FA Authenticator**  
  说明 Esta extensión proporciona códigos de autenticación 2FA en el navegador.  
  版本 1.4.1 大小 1.2 MB  
  好像是法文，但能用。  
  能导出备份，会显示二维码。备份导出为json，内容被加密(openssl enc'd)。  
* **Authenticator**【chrome】  
  【Edge没有】  
  说明 Generate TOTP codes like Google Authenticator.  
  版本 1.0 大小 1.1 MB  
  添加后，不能导出备份，不会显示二维码。  
* **Authenticator App - 2 FA 验证器**  
  说明 使用身份验证器作为您最好的 2 fa 身份验证应用程序。 生成两因素身份验证代码。  
  版本 1.2.0 大小 15.3 MB  
  能导出备份，会显示二维码。备份导出为链接。  
* **两步验证(Authentication) - 2FA**  
  说明 可以让用户方便地获取二次验证的验证码  
  版本 0.0.22 大小 3.0 MB  
  能导出备份，会显示二维码，会给出链接。备份导出为json，内容包含链接。  
* **身份验证器**【Chrome】  
  Authenticator: 2FA Client【Edge】  
  说明 身份验证器用以在浏览器中生成二步认证代码。  
  版本 8.0.1 大小 11.2 MB  
  能导出，会显示二维码。备份导出为链接。  
* **Open Two-Factor Authenticator**【Edge】  
  【chrome没有】  
  描述 一个开源的双因素基于时间的一次性密码（TOTP）认证器，具有SHA-256安全存储。  
  版本 0.3.0 大小 < 1 MB   
  添加后，不能导出备份，不会显示二维码。强制主密码保护，不好用。

## Mozilla Firefox
* 2025-01 记录。  
* **uOTP**  
  A simple OTP manager that doesn't require your data.  
  作者 AYO 版本 1.5.1  
  主页 https://gitlab.com/corentin-pro/uotp  
  能导出备份，不会显示二维码。备份导出为json，内容仅密钥。  
* **ZeeOTP**  
  Generate OTP for a secured access  
  作者 ZeeTim 版本 2.2.1  
  能导出备份，会显示二维码。备份导出为链接。
